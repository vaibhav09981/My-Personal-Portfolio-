Recycle Hub

A community-focused web app that connects users with nearby recycling centers and enables eco-friendly waste pickup scheduling. Built using React and Firebase, Recycle Hub promotes sustainability through a clean UI, real-time data, and educational resources.

Tech Stack

  • TypeScript: Type-safe JavaScript, evident from .ts and tsconfig.json files.
  • Configuration & Dependencies: next.config.js – Custom Next.js configurations; package.json & package-lock.json – Define project dependencies and scripts.
  • Other Notables: index.html – May be used for static fallback or in a public folder (unusual in pure Next.js apps, but can be used); Likely using Node.js as the runtime (standard with Next.js projects).
